# ms-autenticacao
Microserviço de autenticação com NodeJS. Live da DIO

Composição do projeto
Neste projeto Temos alguns Endpoints Base que podem ser extendidos da forma mais adequada para seu contexto.

São eles:

<br>
<h2>Usuários</h2>

<br>

GET /users

GET /users/:uuid

POST /users

PUT /users/:uuid

DELETE /users/:uuid

<br>
<h2>Autenticação</h2>

<br>
POST /token

POST /token/validate
